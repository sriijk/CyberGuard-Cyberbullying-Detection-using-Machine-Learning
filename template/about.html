About Us

This web application aims to detect cyberbullying through Natural Language Processing (NLP) and Machine Learning (ML).

The goal is to raise awareness and provide a practical tool to prevent harmful or offensive language online. Our model is trained to classify input text as either safe or bullying-related, offering users a safer digital experience.

This project is built by Srishti Bhatnagar, Computer Science Engineering (Final Year), with the mission to bring AI-powered protection to real-world problems like cyberbullying.
